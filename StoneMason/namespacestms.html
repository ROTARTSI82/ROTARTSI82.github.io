◆ SSLCacheModeBits

enum stms::SSLCacheModeBits

Simple wrapper around OpenSSL server cache modes (see OpenSSL docs for SSL_CTX_set_session_cache_mode)

Enumerator
eOff 

No session caching takes place. (See OpenSSL docs for SSL_SESS_CACHE_OFF)

eBoth 

Equal to eClient | eServer. (See OpenSSL docs for SSL_SESS_CACHE_BOTH)

eClient 

Cache client sessions. (See OpenSSL docs for SSL_SESS_CACHE_CLIENT)

eServer 

Cache server sessions. This is the default. See SSL_SESS_CACHE_SERVER

eNoAutoClear 

Disable auto clearing of expired sessions. See SSL_SESS_CACHE_NO_AUTO_CLEAR

eNoInternal 

Equal to eNoInternalLookup | eNoInternalStore. See SSL_SESS_CACHE_NO_INTERNAL

eNoInternalLookup 

Don't internally look up sessions in the cache. See SSL_SESS_CACHE_NO_INTERNAL_LOOKUP

eNoInternalStore 

Don't store sessions in the cache internally. See SSL_SESS_CACHE_NO_INTERNAL_STORE
