void stms::GLShaderProgram::autoBindAttribLocs ( const GLchar *  vertShaderSrc) const

Automatically bind vertex attributes to deduced locations. This function would assume that vertex attributes take on the form of attribute [type] [name];

There must be EXACTLY one space between everything and NO leading or trailing whitespace.

Parameters
vertShaderSrcSource code of the vertex shader to deduce attribute locations from.
