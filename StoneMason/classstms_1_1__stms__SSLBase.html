◆ timeoutMs

unsigned stms::_stms_SSLBase::timeoutMs = 1000
protected

The timeout (in milliseconds) for IO operations like send and reading. This is the maximum amount of time we will wait for an IO operation to complete. This timeout is waited for on a thread pool thread, so the main thread won't be affected by this.
